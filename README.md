# AmneziaVPN Telegram Bot

Telegram-бот для удобного управления VPN-профилями [AmneziaVPN](https://amnezia.org). 

Бот предоставляет полноценную панель администратора, поддерживает мультипрофили, имеет встроенное Telegram Mini App и публичный веб-сервис для выдачи конфигураций по секретному ключу.

---

## 🌟 Возможности

**Для пользователей:**
- 🔐 **Мультипрофили:** несколько независимых VPN-профилей на один аккаунт.
- 📥 **Удобный экспорт:** получение конфига текстом (`vpn://…`) или `.vpn`-файлом.
- 🌐 **Mini App:** управление профилями прямо внутри Telegram через удобный веб-интерфейс.
- 🔑 **Автономный доступ:** генерация секретного ключа (`/mykey`) для получения конфига на сайте без участия Telegram.
- 📊 **Мониторинг:** статус профиля и сервера, статистика трафика.

**Для администраторов (`BOT_MODE=admin`):**
- 👥 Управление пользователями (просмотр, бан/разбан, статистика по трафику и онлайну).
- ⏸ Точечное управление профилями (включение/отключение/удаление с синхронизацией).
- 📢 Массовые рассылки и экспорт базы в CSV.
- 🔎 Просмотр всех пиров Amnezia и поиск по ID/имени.

---
